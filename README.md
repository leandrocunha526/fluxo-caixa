# Lançamentos API - Spring Boot

## Execução

```shell
mvn spring-boot:run
```

ou

```shell
./mvnw spring-boot:run
```

Usando Maven

### Com Docker

```shell
docker compoose -f compose.yml up
```

### Testes

```shell
mvn test
```

ou

```shell
./mvnw test
```

## Docs

Você pode conferir as documentações das rotas e realizar requisições em http://localhost:8080/swagger-ui/index.html.
